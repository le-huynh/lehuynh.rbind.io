Question

I am creating a new function in my Rpackage. After running devtools::check(), I get this WARNING:

checking dependencies in R code ... WARNING
'::' or ':::' imports not declared from:
  ‘dplyr’

I would like to correct it.

Answer

Add dplyr to Imports in the DESCRIPTION file.

usethis::use_package("dplyr")
#> Adding dplyr to Imports
#> Refer to functions with dplyr::fun()

Further information

Dependencies: What does your package need?, R Packages by Hadley Wickham and Jenny Bryan
